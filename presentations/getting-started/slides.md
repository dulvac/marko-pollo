## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| :arrow_left: :arrow_right: | Navigate slides |
| **F** | Toggle fullscreen |
| **O** | Overview grid |
| **E** | Open editor |
| **1-9** | Jump to slide |

## Writing Slides

- Separate slides with `---` on its own line
- Use standard **Markdown** for content
- Add code blocks with syntax highlighting
- Include Mermaid diagrams for visuals

## Tips for Great Slides

1. Keep slides **focused** — one idea per slide
2. Use code blocks for **real examples**, not pseudocode
3. Add diagrams to explain **architecture and flow**
4. Use the overview grid (**O**) to check your narrative arc
5. Practice with keyboard shortcuts for smooth delivery :tada:
